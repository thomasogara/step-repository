How I built this portfolio website

Technology Used

This portfolio website's frontend was written using HTML5, CSS3, and JavaScript (version ES6+). Bootstrap, a popular open-source CSS framework, was used to style the elements of the site. The backend was written using Java Servlets.

Data Management

For persistent data storage, I used the Google Cloud Datastore and Blobstore API's. Datastore provides a NoSQl database system with minimal configuration or operation overhead. Blobstore provides a means of both storing and serving files of any type, though my concerns were mostly with storing and serving images.
